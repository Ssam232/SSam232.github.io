## Interactive Excel Dashboard

<div class="row mt-4 align-items-start">
  <div class="col-md-5 mt-4 mt-md-0">
    <p>This interactive Excel tool analyzes athlete performance data across multiple metrics. It includes variables such as speed, strength, and overall performance to show how results change under different conditions.</p>

    <p>Users can interact with the data using slicers to filter by athlete or category. As selections change, the charts and tables update automatically. This allows for direct comparison and helps identify patterns in performance.</p>

    <p>The system is built using Excel formulas and linked visualizations. All charts are generated from the data and respond instantly to user input, demonstrating dynamic computation rather than static display.</p>

    <p><strong>Controls:</strong><br>

    <p><strong>Win condition:</strong><br>
    Reach the goal and complete all 3 levels.</p>

    <p><strong>Lose condition:</strong><br>
    Losing all your lives sends Barry back to the start of the level. Running out of time ends the game.</p>

    <p><strong>What I added:</strong><br>
    I added 3 levels, a Practice Mode for an easier experience, a start screen, a Game Over screen with a Try Again button, and a Win screen with a reset button. I also added sounds, animations, anti-cheat measures, and a short grace period after collisions so the player does not lose multiple lives instantly.</p>

    <p><strong>Variables:</strong><br>
    I used lives, timer, and level variables to control the game.</p>
